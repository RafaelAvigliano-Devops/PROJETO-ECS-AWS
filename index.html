Teste Inicial de DevOps

Este é um projeto de exemplo para testar pipelines de DevOps. O site permite mudar a cor de fundo clicando no botão abaixo.

Objetivos do teste:

  • Configurar infraestrutura na AWS usando Terraform
  • Criar VPC, Internet Gateway e sub-redes em duas zonas de disponibilidade
  • Configurar ECS para deploy de containers
  • Implementar CI/CD para build e deploy automático
  • Testar escalabilidade e monitoramento
